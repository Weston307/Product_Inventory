This program was developed to reflect a product inventory. Products included books, movies, and toys.

Actions included:
- Add a product to the inventory
- Remove a product from the inventory (by sku)
- Display the information for a product (given the sku)
- Display the inventory (by sku)
- Display the inventory (by title)
- Process a sale and display the sale

Classes:
- Menu was the driver which determined user input and displayed appropriate messages.
- Movie, Book, and Toy classes extended Product.
- Inventory manipulated and modified the data structure that held the products.
